Lambda IDE or λIde
this is an ide that was created as an easier alternative to Vim or vi
Here is the list of the basic commands:
choose lang: to choose programming language
inst_comp - propertiesetc - install compiler if you don't have it or edit lambda config 
config commands: add lang: java 
/////////////////////////
the feature of this ide is that it is written in a childish easiest way as possible for people and entuasiasts programmers to edit it
it is a project with an open source code and so: Enjoy and have fun! ^owo^ (:
